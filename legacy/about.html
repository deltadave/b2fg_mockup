About This Tool

Learn more about the D&D Beyond to Fantasy Grounds Character Converter

What is this tool?

The D&D Beyond to Fantasy Grounds Character Converter is a web-based tool that allows you to export your D&D Beyond characters and import them directly into Fantasy Grounds Unity. This saves you time and ensures accuracy when transferring your characters between platforms.

How does it work?

Simply enter your D&D Beyond character ID, and the tool will fetch your character data using D&D Beyond's API and convert it to the XML format that Fantasy Grounds can import. Your character must be set to Public visibility for the tool to access it.

Requirements

  • Your D&D Beyond character must be set to Public
  • Fantasy Grounds Unity
  • A modern web browser
