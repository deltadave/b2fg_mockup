D&D Beyond to Fantasy Grounds Character Converter

D&D Beyond character transfer

Easily tranfer characters created at D&D Beyond to Fantasy Grounds.

Log in to your D&D Beyond Account. Mouse over 'Collections' on the navigation bar and select 'My Characters'. View the character you want to convert. In the address bar, it should look something like: https://www.dndbeyond.com/profile/YourUserName/characters/8675309

Note the character ID. In the above example it's 8675309. Paste this into the 'Character ID' box below and select the format you want. After a moment, the XML will populate.

Character ID




Click 'Download Character' to save the character to your computer.

For instructions on how to import the XML file into Fantasy Grounds, please watch this video by Digital Dungeon Master.
